Carafe

To build the carafe I made half of the figure.

To build the spouth and the back part I used the "Coons patch" by defining the two vertical lines via Bezier's curves and the two horizontal lines by Hermite's curves.


Each circular sector have different heights and it was generated individually, using two horizontal curves: a curve of Hermite that follows the semi-circumference and a Bezier's curve that achieves the upper edge. Then I used a function to generate the surface

function move(p,h,s){
	var p1 = p;
	p1[0] = [p[0][0]+s[0],p[0][1]+s[1],p[0][2]+h];
	p1[1] = [p[1][0]+s[0],p[1][1]+s[1],p[1][2]+h];
	return p1;
}

function side_form(p1,p2,s){
	var c0 = CUBIC_HERMITE(S0)(p1)
	var c1 = CUBIC_HERMITE(S0)(move(p1,0.5,[0,s*0.5,0]))
	var c2 = CUBIC_HERMITE(S0)(move(p1,1.8,[0,s*0.3,0]))
	var c3 = BEZIER(S0)(p2)
	var b2 = MAP(BEZIER(S1)([c0,c1,c2,c3]))(dom2D)
	return b2;
}

The handle was designed by Coons patches.


Then to realize the other half of the figure I rotated the points relative to the y axis through a function. 
function rotate(punti,v){
	var p1 = punti;
	var p1 = p1.map(function(p){return[p[0]*v[0],p[1]*v[1],p[2]*v[2]]});
	return p1;
}
To build the edge at the base I generated a sector and I rotated of PI/5 to fill the entire surface. This is the result obtained in javascript and python.
