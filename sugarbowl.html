Sugar Bowl

Since the very geometric shape of the sugar bowl I decided to build a field for each element and rotate around the z axis.

I have therefore defined the form of a single sector of the sugar bowl, through Hermite and Bezier curves and Coons patches, I then replicated the shape and I colored it of two different colors: black and white. Then this figure is obtained by rotating the volume of the sugar bowl.

I have added the fund to the figure by rotating a circular arc (created using the function arc). I then generated the cover through the lid of the sphere function and a bounded domain to PI/3 in height and PI/6 in width. At last I turned the sector achieved.

var domain_sphere = DOMAIN([[0,PI/3],[-PI/12,PI/12]])([24,36]);
var sphere = function(r){
	return function(v){
	return [r*SIN(v[0])*COS(v[1]),r*SIN(v[0])*SIN(v[1]),r*COS(v[0])]
	}
}
var mapping = sphere(1.3);
var partial_cover = MAP(mapping)(domain_sphere);
var partial_cover_black = COLOR([0,0,0])(partial_cover)
var partial_cover_white = COLOR([1,1,1])(R([0,1])(PI/6)(partial_cover))
var partial_cover_bw = STRUCT([partial_cover_white,partial_cover_black])

var cover = T([2])([1.26])(STRUCT([rotate_form(partial_cover_bw,[0,1],PI/3,5),knob]))
