CUP

Since the very geometric shape of the cup I thought I'd build a field for each element and rotate it around the z axis.

I have therefore created the bottom of the saucer creating a circular sector through the function 'arc' and a rotation. 

function arc(alpha,r,R){
	var domain = DOMAIN([[0,alpha], [r,R]])([36,1]);
	var mapping = function(v){
		var a = v[0];
		var r = v[1];
		return [r*COS(a), r*SIN(a)];
	};
	var model = MAP(mapping)(domain);
	return model;
};

I then defined the form of a single sector of the saucer through the Coons patch, I replicated and I colored one white and the other black.


Turning then the figure you get the saucer.

function rotate_form(partial,axes,angle,n){
	var form = partial
	var f = partial
	for (i=0; i<'n'; i++){
		f = R(axes)(angle)(f)
		form = STRUCT([form,f])
	}
	return form;
}

I followed the same procedure to generate the cup. From a sector I got a cup by turning of PI/3.





I then designed the handle using Coons patches, drawing curves using Bezier curves.
